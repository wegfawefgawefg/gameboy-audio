Chapter 5

Noise is a stream of rough high/low offsets.

For now, noise means the speaker keeps jumping between forward and backward offsets in a pattern that feels random.

The Simple Idea

A tone repeats a neat shape. Noise does not. It keeps choosing high or low offsets. If we fade that noisy motion out quickly, it starts to feel like a drum hit, hat, click, explosion, or texture.

Step 1: A Bit Machine

We need a tiny machine that gives us a changing stream of bits. A bit is just 0 or 1. Later we will turn 0 into one speaker offset and 1 into the other.

sample_rate = 44100
seconds = 1
volume = 0.35
num_samples = seconds * sample_rate

samples = []

# LFSR means linear feedback shift register.
#
# Do not worry about the name yet.
# For now: it is an integer we keep mutating
# so it produces a rough stream of 0/1 bits.
lfsr = 0b111_1111_1111_1111

Step 2: Clock The Noise

The noise value does not have to change every audio sample. We can change it at its own speed. A faster noise clock sounds brighter. A slower noise clock sounds rougher and lower.

noise_clock_hz = 5000
# Change the noise bit 5,000 times per second.
#
# This is much slower than the sample rate:
# 44,100 audio samples per second.
#
# So several audio samples may reuse the same noise value.

timer = 0
current_offset = 1

for sample_index in range(num_samples):
    # Every audio sample moves time forward by 1 / sample_rate.
    timer = timer + 1 / sample_rate

    if timer >= 1 / noise_clock_hz:
        # The noise clock ticked.
        # Time to choose a new high/low offset.
        timer = timer - 1 / noise_clock_hz

Step 3: Make The Next Bit

This is the one weird line. It takes two bits from the current number, XORs them together, shifts the number, and feeds the new bit back into the top. That is enough to make a long not-obviously-repeating bit pattern.

        # Take two bits, xor them, and feed the result back in.
        bit = (lfsr ^ (lfsr >> 1)) & 1
        lfsr = (lfsr >> 1) | (bit << 14)

        # Turn the low bit into a speaker offset.
        #
        # bit set   -> speaker backward
        # bit clear -> speaker forward
        if lfsr & 1:
            current_offset = -1
        else:
            current_offset = 1

Step 4: Shape It Like A Hit

Constant noise is just a hiss. Percussion usually needs a volume shape. Here the noise starts loud and fades to zero over 0.45 seconds.

    # Fade out so it sounds like a tiny drum hit.
    time = sample_index / sample_rate
    fade = max(0, 1 - time / 0.45)

    sample = current_offset * volume * fade
    samples.append(sample)

Flat Code

sample_rate = 44100
seconds = 1
volume = 0.35
num_samples = seconds * sample_rate

samples = []
lfsr = 0b111_1111_1111_1111
noise_clock_hz = 5000
timer = 0
current_offset = 1

for sample_index in range(num_samples):
    timer = timer + 1 / sample_rate

    if timer >= 1 / noise_clock_hz:
        timer = timer - 1 / noise_clock_hz
        bit = (lfsr ^ (lfsr >> 1)) & 1
        lfsr = (lfsr >> 1) | (bit << 14)
        current_offset = -1 if lfsr & 1 else 1

    time = sample_index / sample_rate
    fade = max(0, 1 - time / 0.45)
    samples.append(current_offset * volume * fade)

Game Boy Connection

Channel 4 is the noise channel. It does not play a normal melody pitch like the pulse channels. It clocks an LFSR and turns the bit stream into high/low speaker offsets. The clock settings change whether the noise sounds lower, higher, harder, or softer.
