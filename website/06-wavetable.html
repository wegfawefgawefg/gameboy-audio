How to code GameBoy music.

The third tonal channel is a tiny custom waveform.

What Is A Wavetable?

A wavetable is a small list of offsets that loops. Instead of calculating sine or square from a formula, you draw one cycle once, store it in a table, and read around the table over and over.

Spelled Out

sample_rate = 44100
frequency = 110
volume = 0.35
seconds = 1
num_samples = seconds * sample_rate

# This is one cycle of a tiny triangle-ish wave.
# The real Game Boy wave channel stores 32 4-bit samples.
wave_table = [
    -1.00, -0.75, -0.50, -0.25,
     0.00,  0.25,  0.50,  0.75,
     1.00,  0.75,  0.50,  0.25,
     0.00, -0.25, -0.50, -0.75,
]

samples = []

for sample_index in range(num_samples):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    # Convert phase into a table index.
    table_index = int(phase * len(wave_table))

    sample = wave_table[table_index]
    samples.append(sample * volume)

Game Boy Connection

CH3 is not another pulse channel. It is a small programmable wave. Games can use it for basses, smoother tones, fake instruments, or rough sample-like effects. It is limited, but that limit is exactly why it is interesting.
