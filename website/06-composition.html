Chapter 6

A tiny composition is voices plus noise hits.

Now we can combine the pieces: one pulse melody, one pulse bass/harmony line, and short noise bursts for percussion.

Spelled Out

num_samples = int(1.8 * sample_rate)

samples = []
for _ in range(num_samples):
    samples.append(0)
# Start with silence: one zero offset for every sample slot.

melody = [
    (523.25, 0.00, 0.16),  # C5
    (659.25, 0.20, 0.16),  # E5
    (392.00, 0.40, 0.16),  # G4
    (523.25, 0.60, 0.28),  # C5
]

bass = [
    (261.63, 0.00, 0.38),  # C4
    (392.00, 0.40, 0.38),  # G4
    (261.63, 0.80, 0.38),  # C4
]

for frequency, start, duration in melody:
    add_pulse_note(samples, frequency, start, duration, volume=0.17, duty=0.5)

for frequency, start, duration in bass:
    add_pulse_note(samples, frequency, start, duration, volume=0.14, duty=0.25)

for start in [0.00, 0.40, 0.80]:
    add_noise_hit(samples, start, duration=0.08, volume=0.25)

What Changed?

Nothing magic. We are still filling one output list with offsets. The difference is that multiple little generators add into the same list: melody, bass, and noise hits.
