Chapter 2

Different sounds are different offset patterns.

One voice can use different formulas. We will keep each shape separate, then choose one with a small voice setting.

Square Wave

A square wave chooses between two offsets. The first half of the cycle is forward. The second half is backward.

def square_sample(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    if phase < 0.5:
        return volume
    else:
        return -volume

Triangle Wave

A triangle wave ramps up, then ramps down. It still repeats once per cycle, but it moves in straight lines instead of jumping.

def triangle_sample(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    # phase 0.00 -> -1
    # phase 0.50 -> +1
    # phase 1.00 -> back to -1
    raw = 1 - 4 * abs(phase - 0.5)
    return raw * volume

Selecting A Voice Shape

The simplest selector is not an object. Just write a renderer that accepts the sample function to use.

def render_voice(sample_function, frequency, volume, seconds):
    num_samples = int(seconds * sample_rate)

    samples = []

    for sample_index in range(num_samples):
        sample = sample_function(sample_index, frequency, volume)
        samples.append(sample)

    return samples

square_voice = render_voice(square_sample, 220, 0.35, 1)
triangle_voice = render_voice(triangle_sample, 220, 0.35, 1)
