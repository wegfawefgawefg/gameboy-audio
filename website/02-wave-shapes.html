The Triangulese Wave

A triangle wave goes up to its max, then back down to negative max. Thats pretty much it. 

def triangle(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1
    half_cycle = 0.5

    # left half, line go up
    if phase < half_cycle:
        # frac is 0.0-1.0, where 0.0 is the start of the phase, and 1.0 is the end of the phase.
        frac = phase / half_cycle
        raw = -1 + frac * 2
        # so at the begining of the first phase, raw is -1 + 0 so -1, the bottom of the wave
        # and at the end of the first phase, raw is -1 + 2 so +1, the top of the wave
        # so its a line that goes from -1 to +1
    # right half
    else:
        frac = (phase - half_cycle) / half_cycle
        raw = 1 - frac * 2
        # same thing but down from +1 to -1

    return raw * volume

Sine Wave

A sine wave wont stab ur ears like the square and triangle waves. You wouldnt have heard this on the gameboy, but we're in the future now and you should know this one. Just know if youre trying to make a gameboy sound, use the others. 

import math


def sin(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    return math.sin(phase * 2 * math.pi) * volume

Pulse Wave

The Game Boy does not have a sine channel. The two main melody channels are pulse channels. Remember that caveat I mentioned earlier? Yeah the two main channels arent just square wave channels. A pulse wave is a square wave where the up part and down part can be different widths.

That adjustable width is called duty. A normal square wave is 50 percent duty. The Game Boy pulse channels can do four duty shapes: 12.5 percent, 25 percent, 50 percent, and 75 percent. Same pitch, different bite. At 50% its just a regular square wave. 

def pulse(sample_index, frequency, volume, duty):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    if phase < duty:
        return volume
    else:
        return -volume
GAY BOY

You aren't imagining things, the pitch does seem to change when you change the duty. But duty isnt pitch. With these non sin waves, percieved frequency isnt just the frequency of the wave. If you listen closely you can hear the base frequency, and a second tinny weaker ghost frequency at the wave width of the duty.

On the gameboy you've got 4 settings for duty of each pulse channel. 0 is 12.5 percent, 1 is 25 percent, 2 is 50 percent, and 3 is 75 percent. 1 and 3 sound exactly the same.

Pick and choose

So we've got three wave shapes, and you can make as many as you want, but youll want it to be easy to choose between them. You can make a render function which takes in a voice and params, and that lets your voices be mostly interchangeable. 

import math


# Voice functions

def square(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    if phase < 0.5:
        return volume
    else:
        return -volume


def pulse(sample_index, frequency, volume, duty):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    if phase < duty:
        return volume
    else:
        return -volume


def triangle(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1
    half_cycle = 0.5

    if phase < half_cycle:
        frac = phase / half_cycle
        raw = -1 + frac * 2
    else:
        frac = (phase - half_cycle) / half_cycle
        raw = 1 - frac * 2

    return raw * volume


def sin(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    return math.sin(phase * 2 * math.pi) * volume


# Render one voice into a list of samples.
# params is the stuff that voice needs after sample_index.

def render(sample_function, duration, params):
    num_samples = int(duration * sample_rate)
    voice_samples = []

    for voice_index in range(num_samples):
        sample = sample_function(voice_index, *params)
        voice_samples.append(sample)

    return voice_samples


square_note = render(square, 0.50, [220, 0.25])
gameboy_pulse_note = render(pulse, 0.50, [220, 0.25, 0.25])
triangle_note = render(triangle, 0.50, [330, 0.25])
