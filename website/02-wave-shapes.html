Chapter 2

Different sounds are different offset patterns.

One voice can use different formulas. We will keep each shape separate, then choose one with a small voice setting.

Square Wave

A square wave chooses between two offsets. The first half of the cycle is forward. The second half is backward.

def square(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    if phase < 0.5:
        return volume
    else:
        return -volume

Triangle Wave

A triangle wave ramps up, then ramps down. It still repeats once per cycle, but it moves in straight lines instead of jumping.

def triangle(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1

    # phase 0.00 -> -1
    # phase 0.50 -> +1
    # phase 1.00 -> back to -1
    raw = 1 - 4 * abs(phase - 0.5)
    return raw * volume

Rendering One Voice

A voice function can be passed around like any other value. Pick square or triangle, then render that one voice into its own sample list.

def render(sample_function, frequency, volume, duration):
    num_samples = int(duration * sample_rate)
    voice_samples = []

    for voice_index in range(num_samples):
        sample = sample_function(voice_index, frequency, volume)
        voice_samples.append(sample)

    return voice_samples


square_note = render(square, 220, 0.25, 0.50)
triangle_note = render(triangle, 330, 0.25, 0.50)

Adding Waves Together

If two sample lists happen at the same time, add their offsets together. The speaker gets one final offset per sample, but that final offset contains both waves.

first_voice = square_note[100]
second_voice = triangle_note[100]

mixed_sample = first_voice + second_voice

That is the whole idea behind mixing. Render each sound separately, then sum the samples that land at the same time.

Compositing Sounds Into Output

add places a rendered voice into the larger output list. The start time chooses where the first rendered sample lands.

def render(sample_function, frequency, volume, duration):
    num_samples = int(duration * sample_rate)
    voice_samples = []

    for voice_index in range(num_samples):
        sample = sample_function(voice_index, frequency, volume)
        voice_samples.append(sample)

    return voice_samples


def add(output_samples, start_time, voice_samples):
    start_index = int(start_time * sample_rate)

    for voice_index, sample in enumerate(voice_samples):
        output_index = start_index + voice_index

        if output_index >= len(output_samples):
            break

        # Add into the existing output list.
        # This lets multiple voices overlap.
        output_samples[output_index] = output_samples[output_index] + sample

output = []
for _ in range(int(2 * sample_rate)):
    output.append(0)

square_note = render(square, 220, 0.25, 0.50)
triangle_note = render(triangle, 330, 0.25, 0.50)

add(output, 0.00, square_note)
add(output, 0.00, triangle_note)
