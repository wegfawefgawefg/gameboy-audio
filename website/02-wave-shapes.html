Chapter 2

Different sounds are different offset patterns.

A sine wave moves smoothly. A square wave snaps between two offsets. A triangle wave moves in straight ramps. Same sample list idea, different math.

Sine

Smooth motion. Useful as the cleanest possible starter tone, but not a Game Boy pulse channel sound.

Square

Only two offsets: high and low. This is the important one for Game Boy pulse channels.

Triangle

Moves in straight lines. The Game Boy does not have a triangle channel, but it is useful for understanding wave shapes.

Spelled Out

sample_rate = 44100
frequency = 220
volume = 0.35
seconds = 1
num_samples = seconds * sample_rate

square_samples = []
triangle_samples = []

for sample_index in range(num_samples):
    time = sample_index / sample_rate

    # phase is where we are inside one cycle.
    #
    # phase = 0.00 means the cycle just started.
    # phase = 0.50 means halfway through the cycle.
    # phase wraps back to 0.00 after it reaches 1.00.
    phase = (frequency * time) % 1

    # Square wave:
    # first half of the cycle is forward,
    # second half is backward.
    if phase < 0.5:
        square = 1
    else:
        square = -1

    # Triangle wave:
    # this makes a ramp up, then a ramp down.
    triangle = 1 - 4 * abs(phase - 0.5)

    square_samples.append(square * volume)
    triangle_samples.append(triangle * volume)

Flat Code

sample_rate = 44100
frequency = 220
volume = 0.35
seconds = 1
num_samples = seconds * sample_rate
samples = []

for sample_index in range(num_samples):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1
    sample = 1 if phase < 0.5 else -1
    samples.append(sample * volume)
