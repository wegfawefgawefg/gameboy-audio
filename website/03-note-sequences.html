Chapter 3

A melody is one voice changing notes over time.

Instead of one frequency forever, we render a note for a duration, then render the next note after it.

Spelled Out

sample_rate = 44100

def square_sample(sample_index, frequency, volume):
    time = sample_index / sample_rate
    phase = (frequency * time) % 1
    if phase < 0.5:
        return volume
    return -volume

notes = [
    # frequency, start time, duration
    (261.63, 0.00, 0.20),  # C4
    (329.63, 0.22, 0.20),  # E4
    (392.00, 0.44, 0.20),  # G4
    (523.25, 0.66, 0.30),  # C5
]

seconds = 1.2
samples = [0] * int(seconds * sample_rate)

for frequency, start_time, duration in notes:
    start = int(start_time * sample_rate)
    length = int(duration * sample_rate)

    for i in range(length):
        output_index = start + i
        if output_index >= len(samples):
            break

        samples[output_index] = square_sample(i, frequency, 0.25)

The New Idea

The waveform list is still just offsets. The new trick is placement. A note has a start time and a duration, so we write its samples into one region of the bigger output list.
