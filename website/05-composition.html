How to code GameBoy music.

Bottling It Up

Up to this point we had one off scripts and sketches, but if you want to bottle this up so you can add voices, make music, and do less plumbing, then this might be a way you could do it.

I am not a big fan of OOP and complicated code. I like structs and functions. I think this is pretty bland and you could recreate it in any language. Not saying you have to do it like this, but this is my personal preference since I have a small brain. (Also it makes for better teaching material if anybody can follow along in any language.)

Lets put the note -> freq look up table in a file with the note consts. 

# notes.py

names = ["C", "C#", "D", "D#", "E", "F",
         "F#", "G", "G#", "A", "A#", "B"]

# Build the note name -> frequency lookup table.
note_freq = {}
for midi_note in range(24, 96):
    name = names[midi_note % 12]
    octave = (midi_note // 12) - 1
    frequency = 440 * (2 ** ((midi_note - 69) / 12))
    note_freq[name + str(octave)] = frequency

And you can put your voices in another file. 

# voices.py

import random

from notes import note_freq


def square(sample_index, sample_rate, state, note, duty, volume):
    frequency = note_freq[note]
    time = sample_index / sample_rate
    phase = (time * frequency) % 1

    if phase < duty:
        return volume

    return -volume


def noise(sample_index, sample_rate, state, hold, volume):
    if sample_index % hold == 0:
        state[0] = random.uniform(-1, 1)

    return state[0] * volume

Next we'll make some utility functions we cant get away from:

  • Taking a voice and params and spitting out a list of samples.
  • Compositing a list of samples into another larger buffer of samples.
# audio_rendering.py

sample_rate = 44100


# Render one event into its own temporary sample list.
def render(voice, seconds, fade, params):
    num_samples = int(seconds * sample_rate)
    state = [0]

    samples = []
    for i in range(num_samples):
        value = voice(i, sample_rate, state, *params)
        samples.append(value)

    for i in range(num_samples):
        percent_done = i / num_samples
        fade_amount = (1 - percent_done) ** fade
        samples[i] = samples[i] * fade_amount

    return samples


# Add a rendered sample list into the output list at a position.
def add(output, rendered, start_sample):
    needed_samples = start_sample + len(rendered)
    while len(output) < needed_samples:
        output.append(0)

    for i in range(len(rendered)):
        output[start_sample + i] += rendered[i]


# Convert musical steps into samples, then render and add the event.
# really this is here for convenience. 
# You could just render by start position in seconds and length in seconds, 
# but then you cant adjust the speed of the song. Youd have to go and hand edit every note start and length.
# So we divide time up into units of beats, and work with that.
# You can take this a step further to make it even more convenient later, but this is a start.
def put(output, voice, start_step, length_steps, fade, params, step_seconds):
    seconds = length_steps * step_seconds
    start_sample = int(start_step * step_seconds * sample_rate)
    rendered = render(voice, seconds, fade, params)
    add(output, rendered, start_sample)


# Render every track into one final output list.
def render_song(song, step_seconds):
    output = []

    for track in song:
        for voice, start, length, fade, *params in track:
            put(output, voice, start, length, fade, params, step_seconds)

    return output

Heres how youd use it. 

from audio import play
from audio_rendering import render_song
from notes import C3, C4, E3, E4
from voices import noise, square

bpm = 148
step_seconds = 60 / bpm / 4

# (voice, start, length, fade, ...voice_params)
track_1 = [
    (square, 0, 6, 0, E3, 0.50, 0.10),
    (square, 8, 4, 0, C3, 0.50, 0.10),
    (square, 12, 4, 0, E3, 0.50, 0.10),
]

track_2 = [
    (square, 0, 6, 0, C4, 0.25, 0.12),
    (square, 8, 10, 0, E4, 0.25, 0.12),
]

drum_track = [
    (noise, 0, 12, 0.20, 42, 0.22),
    (noise, 24, 1, 0.05, 8, 0.18),
    (noise, 28, 1, 0.06, 18, 0.16),
]

song = [
    track_1,
    track_2,
    drum_track,
]

samples = render_song(song, step_seconds)
play(samples)

step_seconds is the grid size. start is the cursor position on that grid.

The Shape

Render one sound. Add it into the song. Repeat until the song is full. Thats the whole machine.
